📜 Правила сервера FUNWORLD

1. Основные правила

  • 1.1 Уважайте всех игроков сервера.
  • 1.2 Запрещены оскорбления, унижения и токсичное поведение.
  • 1.3 Запрещён спам и флуд в чате.
  • 1.4 Запрещена реклама других серверов без разрешения администрации.
  • 1.5 Нельзя выдавать себя за администрацию.

2. Игровые правила

  • 2.1 Запрещено использовать читы, моды дающие преимущество (X-Ray, KillAura и т.д.).
  • 2.2 Запрещено использовать баги и дюпы сервера.
  • 2.3 Гриферство запрещено (разрушение чужих построек).
  • 2.4 Нельзя воровать вещи у игроков без разрешения.
  • 2.5 Запрещено мешать игре другим игрокам.

3. Правила чата

  • 3.1 Не использовать капс (ПИСАТЬ ВОТ ТАК).
  • 3.2 Запрещён мат и оскорбления.
  • 3.3 Не флудить одинаковыми сообщениями.
  • 3.4 Запрещено обсуждение политики и конфликтов.

4. Правила для администрации

  • 4.1 Администратор обязан помогать игрокам.
  • 4.2 Запрещено использовать админку в личных целях.
  • 4.3 Администратор должен соблюдать правила сервера.
  • 4.4 Запрещено выдавать наказания без причины.

5. Наказания

  • 5.1 Предупреждение.
  • 5.2 Мут (запрет писать в чат).
  • 5.3 Кик (выгнать с сервера).
  • 5.4 Бан (временный или навсегда).

✅ Играя на сервере FUNWORLD, вы автоматически соглашаетесь с правилами.
